## 📚 Stage 5: 术语表更新阶段（新增）

### 目的
确保所有翻译的术语都被记录在统一的术语表中，便于后续翻译保持一致性。

### 操作步骤
1. **提取新术语**
   - 从本次翻译的文件中提取所有 `name` / `name_en` 对照
   - 识别新的或变更的术语

2. **更新术语表**
   - 文件位置：`translations/GLOSSARY.md`
   - 按类别添加/更新术语
   - 包含：属性、技能、职业、子职业、种族、装备、法术等

3. **检查一致性**
   - 确保同一术语在不同文件中译名统一
   - 如发现不一致，标记并修复

### 术语表结构
```markdown
## 类别名称

| 英文 | 中文 | 备注 |
|-----|------|-----|
| Term Name | 术语译名 | 可选说明 |
```

### 提交内容
术语表更新应与翻译文件一起提交：
```bash
git add translations/GLOSSARY.md
```

## 📊 Token 统计方案

| 项目 | 统计方式 | 估算依据 |
|-----|---------|---------|
| **输入 Token** | 源文件字符数 ÷ 4 | 英文平均 4 字符/Token |
| **输出 Token** | 翻译文件字符数 ÷ 2 | 中文平均 2 字符/Token |
| **总计** | 输入 + 输出 | 估算值，误差 ±20% |

> ⚠️ 注：实际 Token 消耗以 API 账单为准

## ⏱️ 等待机制详细规则

```
开始等待
    │
    ▼
┌──────────┐
│ 等待 2分钟 │◄───────┐
└──────────┘        │
    │               │
    ▼               │
时间到              │
    │               │
    ▼               │
询问："是否继续等待？  │
      回复则翻倍时间   │
      不回复则直接修复"│
    │               │
    ├── 用户回复 ────┘（等待时间 ×2，回到等待）
    │
    └── 2分钟内无回复
            │
            ▼
        直接执行修复
```

### 等待时间递增表

| 轮次 | 等待时间 | 触发条件 |
|-----|---------|---------|
| 第1轮 | 2 分钟 | 默认初始等待 |
| 第2轮 | 4 分钟 | 用户回复继续等待 |
| 第3轮 | 8 分钟 | 用户再次回复继续等待 |
| 第4轮 | 16 分钟 | 用户再次回复继续等待 |
| ... | 翻倍 | 每次用户回复 |

## 📁 文件命名规范

### 翻译文件
- 位置：`src/2014-zh/`
- 格式：`5e-SRD-{Name}.json`
- 示例：`5e-SRD-Classes.json`

### 审查报告
- 位置：`translations/review/`
- 格式：`5e-SRD-{Name}.review.md`
- 示例：`5e-SRD-Classes.review.md`

### 术语映射
- 位置：`translations/`
- 格式：`core-{category}.json` 或 `{category}-map.json`
- 示例：`core-skills.json`, `races-map.json`

---

## 📝 翻译格式规范

### 必须遵循
1. **`index`** - 永不翻译，保持原样（API 兼容性）
2. **`url`** - 永不翻译，保持原样
3. **`name`** - 翻译为中文，同时添加 `name_en` 保留英文
4. **`desc`** - 翻译为中文，同时添加 `desc_en` 保留英文

### 可选字段
- `type` / `type_en` - 类型（如 Standard/标准）
- `script` / `script_en` - 文字（如 Common/通用文）
- `ability` / `ability_zh` - 属性缩写和中文

---

## 🔍 审查要点

1. **术语一致性** - 是否使用统一译名
2. **翻译准确性** - 有无错译、漏译
3. **中英混杂** - 是否出现 "an 奥术法器" 等问题
4. **格式规范性** - 是否遵循 `name` + `name_en` 格式
5. **完整性** - 所有字段是否都已翻译
