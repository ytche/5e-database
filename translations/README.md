## 📝 翻译原则

1. **index 和 url 字段**：永不翻译，保持原样（API 兼容性）
2. **name 字段**：`中文 (英文缩写)` 格式，如 `力量 (STR)`
3. **full_name 和 desc**：完整翻译为中文，同时保留 `full_name_en` 和 `desc_en`
4. **专业术语**：优先使用 D&D 官方/社区通用译名
