## ❌ 错误/遗漏

| 位置 | 问题描述 | 建议修复 |
|------|----------|----------|
| `all-armor` → `reference.name` | 该字段值为英文 "Armor"，未翻译 | 应改为 "护甲"，与其他 `reference.name` 保持一致 |

### 详细说明

**问题位置**：`index: all-armor` 条目的 `reference` 对象

**当前内容**：
```json
"reference": {
  "index": "armor",
  "name": "Armor",
  "url": "/api/2014/equipment-categories/armor"
}
```

**建议修复**：
```json
"reference": {
  "index": "armor",
  "name": "护甲",
  "url": "/api/2014/equipment-categories/armor",
  "name_en": "Armor"
}
```

**问题原因**：
- 此条目遗漏了 `reference.name` 的翻译
- 同时遗漏了 `reference.name_en` 字段（其他条目均有）
- 该条目主对象的翻译是完整的（`name`: "所有护甲", `name_en`: "All armor"）
