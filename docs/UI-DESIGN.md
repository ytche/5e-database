> 版本: 1.0

## 一、整体视觉风格

### 1.1 设计哲学
- **核心概念**: "古老的法术书 meets 现代数字工具"
- **氛围**: 神秘、沉浸、专业
- **暗色优先**: 保护眼睛，适合长时间查阅，营造地下城探险感

### 1.2 色彩方案

#### 主色调 (Primary Colors)
```css
/* 背景层级 */
--bg-primary: #0F0F13;       /* 主背景 - 深邃夜空 */
--bg-secondary: #1A1A22;     /* 次级背景 - 卡片/面板 */
--bg-tertiary: #252532;      /* 三级背景 - 悬停/激活状态 */
--bg-elevated: #2D2D3D;      /* 提升层级 - 弹窗/下拉菜单 */

/* 强调色 - 法术学派配色 */
--accent-evocation: #FF6B35;     /* 塑能 - 橙红色 (火焰/闪电) */
--accent-conjuration: #4ECDC4;   /* 咒法 - 青绿色 (召唤) */
--accent-abjuration: #45B7D1;    /* 防护 - 天蓝色 (护盾) */
--accent-illusion: #9B59B6;      /* 幻术 - 紫色 (幻象) */
--accent-divination: #F1C40F;    /* 预言 - 金色 (预知) */
--accent-enchantment: #E91E63;   /* 惑控 - 玫红色 (心智) */
--accent-necromancy: #8B4513;    /* 死灵 - 深棕色 (死亡) */

/* 功能色 */
```

### 1.3 字体系统

#### 字体栈
```css
/* 中文标题 - 书法感 */
--font-display: "Noto Serif SC", "Source Han Serif SC", "STSong", serif;

/* 中文正文 - 清晰可读 */
--font-body: "Noto Sans SC", "Source Han Sans SC", "PingFang SC", "Microsoft YaHei", sans-serif;

/* 英文/数字 - 等宽数据展示 */
--font-mono: "JetBrains Mono", "Fira Code", "Consolas", monospace;

/* D&D 风格装饰字体（可选） */
--font-fantasy: "Cinzel", "Trajan Pro", "Noto Serif SC", serif;
```

#### 字号规范
```css
/* 标题层级 */
--text-hero: 3rem;        /* 48px - 首页大标题 */
--text-h1: 2.25rem;       /* 36px - 页面标题 */
--text-h2: 1.75rem;       /* 28px - 区块标题 */
--text-h3: 1.375rem;      /* 22px - 卡片标题 */
--text-h4: 1.125rem;      /* 18px - 小标题 */

/* 正文 */
--text-body: 1rem;        /* 16px - 默认正文 */
--text-small: 0.875rem;   /* 14px - 辅助文字 */
--text-xs: 0.75rem;       /* 12px - 标签/徽章 */

/* 行高 */
--leading-tight: 1.25;
--leading-normal: 1.6;
--leading-relaxed: 1.75;
```

#### 字体样式
- **标题**: font-weight: 600-700, letter-spacing: 0.02em
- **正文**: font-weight: 400, line-height: 1.7 (中文需要更大行高)
- **数据/数值**: font-family: var(--font-mono), color: var(--accent-primary)

### 1.4 图标系统
- **主要**: Lucide React (简洁现代)
- **备选**: FontAwesome (更多奇幻主题图标)
- **尺寸**: 16px (sm), 20px (md), 24px (lg), 32px (xl)

## 二、页面结构

### 2.1 站点地图

```
/
├── 首页 (Home)
│   └── 快速导航 + 全局搜索
│
├── 法术 (Spells)
│   ├── 列表页 - 筛选器 + 法术网格
│   └── 详情页 - 法术卡片完整展示
│
├── 职业 (Classes)
│   ├── 列表页 - 职业卡片
│   └── 详情页 - 职业特性 + 子职业
│
├── 种族 (Races)
│   ├── 列表页
│   └── 详情页
│
├── 怪物图鉴 (Monsters)
│   ├── 列表页 - 筛选器 + 怪物网格
│   └── 详情页 - 属性块 + 动作列表
│
├── 装备 (Equipment)
│   ├── 列表页
│   └── 详情页
│
├── 魔法物品 (Magic Items)
│   ├── 列表页 - 按稀有度筛选
│   └── 详情页
│
├── 背景 (Backgrounds)
│   ├── 列表页
│   └── 详情页
│
└── 专长/特性 (Feats/Features)
    └── 列表/详情
```

### 2.2 导航结构

#### 顶部导航栏 (Navbar)
```
[Logo]  D&D 5e 中文数据库                    [搜索框]  [主题切换]
----------------------------------------------------------------
[法术] [职业] [种族] [怪物] [装备] [魔法物品] [背景] [专长] [更多 ▼]
```

#### 移动端导航
- 汉堡菜单 + Logo + 搜索图标
- 展开的侧边抽屉菜单

### 2.3 路由设计
```
/spells              - 法术列表
/spells/:index       - 法术详情 (如 /spells/fireball)

/classes             - 职业列表  
/classes/:index      - 职业详情

/monsters            - 怪物列表
/monsters/:index     - 怪物详情

/items               - 装备列表
/items/:index        - 装备详情

/magic-items         - 魔法物品列表
/magic-items/:index  - 魔法物品详情
```
