  box-shadow: 0 8px 24px rgba(201, 162, 39, 0.15);

  transform: translateY(-1px);

}

## 响应式断点对照

### 桌面端 (≥1280px)
- 卡片网格: 4列
- 导航: 完整水平展示
- 详情页: 左右两栏布局

### 平板横屏 (1024px)
- 卡片网格: 3列
- 导航: 完整水平展示
- 详情页: 左右两栏布局

### 平板竖屏 (768px)
- 卡片网格: 2列
- 导航: 部分折叠或汉堡菜单
- 详情页: 单栏堆叠

### 移动端 (<768px)
- 卡片网格: 1列
- 导航: 汉堡菜单
- 详情页: 单栏堆叠
- 筛选: 底部抽屉
