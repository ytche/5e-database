#### 步骤 7：起始装备 (`/builder/equipment`)

**页面说明**：查看职业和背景提供的起始装备，使用起始金币购买额外装备，或完成特定职业的装备选择。

**适用场景**：
- 职业/背景提供了装备选择（如战士选择武器类型）
- 使用起始金币购买额外装备
- 非法术职业通常有此步骤，法术职业可能跳过或简化

---

**API 数据需求**：

```typescript
interface EquipmentData {
  // 职业和背景已提供的装备
  startingEquipment: {
    class: EquipmentItem[];    // 职业装备
    background: EquipmentItem[]; // 背景装备
  };
  
  // 起始金币（根据职业不同）
  startingGold: number;
  
  // 待完成的装备选择（如有）
  pendingChoices?: EquipmentChoice[];
  
  // 可购买的装备列表
  shopInventory: ShopItem[];
}

interface EquipmentItem {
  index: string;
  name: string;
  quantity: number;
  description?: string;
}

interface EquipmentChoice {
  index: string;
  description: string;       // 如"选择一种近战武器"
  options: EquipmentOption[];
  choose: number;
}

interface ShopItem {
  index: string;
  name: string;
  cost: number;              // 价格（金币）
  costUnit: 'cp' | 'sp' | 'gp' | 'pp';  // 铜/银/金/铂金币
  weight?: number;           // 重量（磅）
  description: string;
  category: 'weapon' | 'armor' | 'gear' | 'tool' | 'mount' | 'service';
}

// 购买结果
interface EquipmentPurchase {
  choices: {                 // 完成的装备选择
    [choiceIndex: string]: string;
  };
  purchased: {               // 购买的装备
    itemIndex: string;
    quantity: number;
  }[];
  remainingGold: number;     // 剩余金币
}
```

**交互说明**：

1. **已获装备卡片**：
   - 展示职业和背景提供的固定装备
   - 按类别分组显示
   - 只读展示，不可修改

2. **装备选择**（如有）：
   - 部分职业需要完成装备选择
   - 如战士：选择近战武器、选择远程武器
   - 单选或多选形式
   - 必须完成选择才能下一步

3. **装备商店**（可选）：
   - 显示剩余金币（根据职业起始金币不同）
   - 战士/野蛮人：无金币（装备已给全）
   - 法师：无金币（装备已给全）
   - 盗贼：无金币但有盗贼工具
   - 部分变体规则可能有起始金币
   
   - **装备分类**：武器、护甲、冒险装备、工具等
   - **价格显示**：gp/sp/cp，买不起的灰色显示
   - **点击购买**：添加到购物车
   - **数量调整**：可购买多个

4. **已购装备**：
   - 显示本次购买记录
   - 实时计算花费和剩余金币
   - 可清空购物车

5. **验证**：
   - 如有待选装备，必须完成选择
   - 商店购买为可选
   - 金币不能为负数

**说明**：

此步骤主要是为了：
1. 让玩家确认已获得哪些装备
2. 完成职业装备中需要选择的选项
3. （可选）用起始金币购买额外装备

大部分职业的起始装备是固定的，此页面可以快速跳过。只有战士、游侠等需要选择武器类型的职业才需要交互。
