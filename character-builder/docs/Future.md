## 🏪 装备商店

**关联页面**: 步骤7 起始装备 (`/builder/equipment`)

**当前状态**: 第一版仅展示已有装备和完成装备选择，不实现商店购买

**功能描述**:
- 使用起始金币购买额外装备
- 装备分类浏览（武器、护甲、冒险装备、工具等）
- 实时显示剩余金币
- 买不起的装备灰色显示
- 购物车功能（数量调整、清空）

**数据需求**:
```typescript
interface ShopItem {
  index: string;
  name: string;
  cost: number;
  costUnit: 'cp' | 'sp' | 'gp' | 'pp';
  weight?: number;
  description: string;
  category: 'weapon' | 'armor' | 'gear' | 'tool' | 'mount' | 'service';
}
```

**5e起始金币规则**（变体规则）:
- 战士: 5d4×10 gp (平均125 gp)
- 法师: 4d4×10 gp (平均70 gp)
- 其他职业以此类推

**优先级**: P2（后续版本考虑）

## 🎲 骰子掷骰属性分配

**关联页面**: 步骤4 分配属性 (`/builder/abilities`)

**当前状态**: 第一版只实现27点购点法

**功能描述**:
- 4d6 drop lowest 掷骰模式
- 显示掷骰结果
- 将掷出的数值分配到六个属性
- 支持重新掷骰

**掷骰规则**:
- 掷4个d6，去掉最低的1个
- 重复6次，得到6个数值
- 将这6个数分配到力量、敏捷、体质、智力、感知、魅力

**优先级**: P2（后续版本考虑）
